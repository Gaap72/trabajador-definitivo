# Reglas de Trabajo - Gemini CLI & Antigravity (Genaro Aurelio)

## ⚡ Antigravity Deep Automation
Se han implementado herramientas de automatización profunda para optimizar el flujo de trabajo:

| Alias | Función | Descripción |
| :--- | :--- | :--- |
| `ag` | **Antigravity CLI** | Acceso directo al motor de Antigravity (chat, tunneling, extensions). |
| `subir` | **GitHub Rápido** | `git add .`, `git commit`, `git push`. Inicializa el repo si no existe. |
| `build` | **Auto-Build** | Detecta el tipo de proyecto (.NET/Node.js) y lo compila automáticamente. |
| `status` | **Health Check** | Verifica el estado del entorno y las herramientas. |
| `db-setup` | **Database Init** | Configura automáticamente la base de datos SQL para el proyecto Ventas. |

## 🛠️ Entorno de Trabajo
- Preferir **Cursor** o **Windsurf** si están disponibles.
- En `programacion`, utilizar estándares C# / .NET de alto rendimiento.

## 🧪 Validación
- Cada cambio debe ser validado con `status` o comandos de terminal específicos antes de finalizar.
